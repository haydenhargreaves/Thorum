Module row

structs

ROW

Properties

  • chars: [dynamic]u8
    • Text content stored in the row.

procedures

row_insert_char

Insert a character into a ROW. This will be used to enter text anywhere in the editor. If an invalid index, less than 0, or greater than or equal to the length is provided, an out of bounds error will be returned. Also, on the off chance allocation fails, an allocation failed error will be returned.

Params

  • row: ^row.ROW
    • The row you wish to insert a character into.
  • pos: int
    • Position to insert the character at (0-indexed).
  • char: u8
    • Character to insert, stored as a byte (unsigned int 8).

Returns

  • row.ROW_ERROR
    • If an error occurs, it will be returned. Otherwise, this value will be nil

row_remove_char

Remove a character from a ROW. This is how the editor will handle deleting characters. If an invalid index, less than 0, or greater than or equal to the length is provided, an out of bounds error will be returned.

Params

  • row: ^row.ROW
    • The row you wish to remove a character from.
  • pos: int
    • Position to remove the character from (0-indexed).

Returns

  • row.ROW_ERROR
    • If an error occurs, it will be returned. Otherwise, this value will be nil
